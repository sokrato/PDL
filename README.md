1. 电子图书馆，已归档的文件不允许编辑，只能添加新版本
2. 不同的版本，文档类型，如何归集？

图书，电影，电视剧，音乐，照片，个人归档文件

出版物有出版商；同一个版本，肯定是同一个出版商

图书有不同版本，每个版本有不同文件格式

电影有不同文件格式、清晰度

电视剧有Seasons，season中的每一集有不同的清晰度、文件格式
